# Ghid rapid: cum să publici și să trimiți invitațiile personalizate

Această pagină este deja gata pentru personalizare: nu ai nevoie de un fișier HTML separat pentru fiecare persoană.

## 1. Publică pagina pe un site gratuit
Alege una dintre opțiunile simple:
- GitHub Pages
- Netlify
- Vercel

Pași:
1. Încarcă tot proiectul în repository-ul tău.
2. Activează Pages / Netlify / Vercel pentru folderul rădăcină.
3. Asigură-te că fișierul principal este `invitatie.html`.
4. Copiază URL-ul public, de exemplu:
   `https://nume-utilizator.github.io/invitatie.html`

## 2. Creează linkuri personalizate pentru fiecare persoană
Folosește parametrul `name` pentru numele persoanei.

Exemple:
- `https://YOUR-DOMAIN/invitatie.html?name=Andrei`
- `https://YOUR-DOMAIN/invitatie.html?name=Ioana`
- `https://YOUR-DOMAIN/invitatie.html?name=Marian&gender=f`
- `https://YOUR-DOMAIN/invitatie.html?name=Andrei&message=Text%20personalizat%20pentru%20el`

## 3. Trimite linkurile prin WhatsApp / Messenger / Telegram
1. Deschide linkul în browser.
2. Verifică că numele și mesajul apar corect.
3. Copiază linkul în chat și trimite-l persoanei.

## 4. Ce poți personaliza
- `name` → numele afișat în pagină
- `gender` → ajută la mesajul implicit
- `message` sau `note` → scrie un mesaj special pentru persoana respectivă

## 5. Format recomandat pentru lista ta
Păstrează o listă simplă astfel:
- Andrei → `...?name=Andrei`
- Ioana → `...?name=Ioana`
- Marian → `...?name=Marian&gender=f`
- etc.

## 6. Test rapid înainte de trimitere
Deschide un link și verifică:
- titlul paginii
- numele persoanei
- mesajul personalizat
- că pagina se încarcă fără erori

Dacă vrei, poți să copiezi direct aceste linkuri într-un document de note și să le trimiți rapid tuturor.
